# CSTORE - Mini Katalog Uygulaması

Bu proje, Flutter eğitimi kapsamında geliştirilmiş bir mini katalog uygulamasıdır. Kullanıcılar Apple ürünlerini listeleyebilir, detaylarını görüntüleyebilir ve sepete ekleme simülasyonu yapabilir.

## 🚀 Kullanılan Flutter Sürümü
Flutter 3.16.0 veya üzeri
